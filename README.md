## マジックナンバーとは何でしょうか

8=2×(2×2)=2×(1+3)

32=2×(4×4)=2×(1+3+5+7)

50=2×(5×5)=2×(1+3+5+7+9)

72=2×(6×6)=2×(1+3+5+7+9+11)
